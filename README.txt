Used to extract and convert PS2 game ace combat radio & music voice.

Supported:
- Ace combat 4
- Ace combat 5
- Ace combat 0

Usage:
1. copy /bin/radioee.pac or /bin/radioej.pac or /bin/us/bgm.pac (for
soundtrack) from ace combat 5 iso to current directory
2. run extract_radio.py radioee.pac. It will dump all radio npsf into rip dir.
3. run lanuch_vgstream.py. It will convert npsf into wav.
